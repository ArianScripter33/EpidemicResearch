# JULES ISSUE 1 — DGE Morbilidad: Extract 2018-2024 Data URLs

## Priority: HIGH | Label: `jules` | Wave: 1.5

---

## Project Context

**Project:** Ganado Saludable — Academic epidemic research project (Licenciatura en Ciencias de Datos para Negocios, UNRC).

**Disease:** Fiebre Aftosa (FMD) — using Tuberculosis Bovina as calibration proxy because Mexico is FMD-free since 1954.

**What we need:** Morbidity data (casos por enfermedad por estado) from Mexico's Dirección General de Epidemiología (DGE) for years 2018–2024.

**Why it matters:** This data feeds:
- ANOVA test (difference in food intoxication A05 by state/channel)  — slide 9 of our 15-slide presentation
- Cross-correlation TB animal (SENASICA) ↔ TB humana (DGE) by state — validates our proxy strategy
- XGBoost features for predicting intoxication events

**What we already have:** DGE anuarios 2015, 2016, 2017 (CSV via ZIP). Those produced 384 rows (288 TB + 96 food intox A05). We need more recent data to strengthen the analysis.

**CIE-10 codes we need:** A15-A19 (tuberculosis, all forms), A05 (intoxicacion alimentaria bacteriana).

---

## The Problem

The DGE page `https://www.gob.mx/salud/documentos/datos-abiertos-152127` only lists CSV ZIPs for 2015-2017.

However, there is a JavaScript-rendered anuario at:
`https://epidemiologia.salud.gob.mx/anuario/html/morbilidad_nacional.html`

This page has a menu with items for years 2018, 2019, 2020, 2021, 2022, 2023, 2024 (found in JS variables `A_2018`, `A_2019`, ... `A_2024`). The links are generated dynamically in JavaScript so we can't see the actual URLs without a browser.

A second page was also found open at:
`https://epidemiologia.salud.gob.mx/anuario/2022/morbilidad/nacional/...`
(403 on directory listing but individual files may be accessible)

Search the web for:
1. `"epidemiologia.salud.gob.mx" "anuario" 2022 CSV download morbilidad`

4. `"datos_abiertos" "Anuario_2018" OR "Anuario_2019" OR "Anuario_2020" site:gob.mx`

### Step 2: Inspect the JS-rendered page
Navigate to:
`https://epidemiologia.salud.gob.mx/anuario/html/morbilidad_nacional.html`

Find the actual download links for each year 2018-2024 by:
- Clicking each year in the menu
- Inspecting what URL is loaded in any iframe or navigated to
- Looking at network requests (XHR/fetch) triggered by the year selector

### Step 3: Try known URL patterns
The 2015-2017 ZIPs were at:
`https://epidemiologia.salud.gob.mx/anuario/datos_abiertos/Anuario_{year}.zip`

Try variations for 2018-2024:
- `https://epidemiologia.salud.gob.mx/anuario/{year}/datos_abiertos/Anuario_{year}.zip`
- `https://epidemiologia.salud.gob.mx/anuario/{year}/datos_abiertos/t{year}.zip`
- `https://epidemiologia.salud.gob.mx/anuario/{year}/morbilidad/datos_abiertos.zip`

### Step 4: Check alternative DGE portals
- `https://www.sinave.gob.mx/`
- `https://epidemiologia.salud.gob.mx/`
- Search GitHub for scrapers that have already solved this (search: `DGE morbilidad scraper python`)

## Expected Output

Write a Python script at `src/extractors/dge_2018_2024.py` that:
1. Downloads morbidity data for years 2018-2024
2. Filters for CIE-10 codes starting with A15, A16, A17, A18, A19 (tuberculosis) and A05 (food intoxication)
3. Saves to `data/raw/dge_morbilidad_{year}.csv`
4. Uses `latin1` or `utf-8` encoding (GovMX CSVs are often latin1)
5. Has same interface as existing `src/extractors/dge_morbilidad.py` (already in the repo)

If the data is behind JS and cannot be scripted directly, write a detailed README comment explaining exactly how to manually download each year and what the URL pattern is.
